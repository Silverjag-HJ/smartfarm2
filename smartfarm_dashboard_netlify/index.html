CSV Upload Dashboard

스마트팜 생육 데이터 대시보드

CSV 파일을 업로드하면 수치형 데이터 기초통계, 목록형 데이터 빈도수, 분석 주제별 차트를 자동 생성합니다.

1. CSV 파일 업로드

예시 컬럼: 시도, 작물명, 측정일자, 평균기온(℃), 평균습도(%), CO2농도(ppm), 일사량(W/㎡), 생육지수

컬럼 건수 평균 최소 중앙값 최대 표준편차

4. 목록형 데이터 빈도수

5. 분석 주제 4개

① 작물별 생육지수 비교

② 측정일자별 생육지수 변화

시간 흐름에 따른 생육지수 변화를 확인합니다.
